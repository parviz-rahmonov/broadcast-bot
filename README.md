# 📬 Broadcast Bot

Telegram-бот для рассылки сообщений с админ-панелью. Написан на Python + aiogram 3.

---

## ⚙️ Функции

- `/start` — регистрирует пользователя в `users.csv`
- `/admin` — открывает панель администратора
- 📊 Статистика — количество пользователей
- 📨 Рассылка — отправка текста, фото или видео всем пользователям
- 🔄 Обновить — обновить данные в панели
- ◀️ Назад / ❌ Закрыть — навигация по панели

---

## 📁 Структура

```
├── main.py       # основной код бота
├── config.py     # токен и admin_id
└── users.csv     # база пользователей (создаётся автоматически)
```

---

## 🚀 Запуск

```bash
pip install aiogram
python main.py
```

---

## 🔧 Настройка

Откройте `config.py` и заполните:

```python
BOT_TOKEN = "токен от @BotFather"
ADMIN_ID  = 123456789  # ваш id — узнать у @userinfobot
```
